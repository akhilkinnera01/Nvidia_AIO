GPU Management (nvidia-smi)

nvidia-smi -L # List GPUs & UUIDs nvidia-smi -q -d MEMORY # Query specific info nvidia-smi topo -m # Check topology (NVLink) nvidia-smi dmon # Monitor device stats nvidia-smi -r # Reset GPU (requires admin)

MIG (A100/H100) Configuration

nvidia-smi -i 0 -mig 1 # Enable MIG mode nvidia-smi mig -lgip # List GPU profiles nvidia-smi mig -cgi 19,19 # Create GI (e.g. 1g.5gb) nvidia-smi mig -cci # Create CIs on GIs nvidia-smi mig -dgi # Destroy all GIs
Profiles:
1g.5gb (slice=7)
2g.10gb (slice=3)
3g.20gb (slice=2)
7g.40gb (slice=1, full)

Slurm Workload Manager

sinfo # Show partition state squeue # List running jobs sbatch --gres=gpu:2 job.sh # Submit batch job srun --gres=gpu:1 -p debug --pty bash # Interactive scontrol show job ID # Inspect specific job scontrol update NodeName=n1 State=DRAIN # Admin
gres.conf: Must define Name=gpu Type=tesla File=/dev/nvidiaX

Kubernetes & GPU Operator

kubectl get nodes -o wide # Check node IP/status kubectl get pods -n gpu-operator # Check operator pods kubectl logs -n gpu-operator -l app=nvidia-driver-daemonset kubectl describe node | grep nvidia.com/gpu
Debug: Check NVIDIA Device Plugin for resource advertising issues.

DCGM (Data Center GPU Manager)

dcgmi discovery -l # List GPUs dcgmi diag -r 1 # Short diagnostic (1 min) dcgmi health -c # Check health watches dcgmi stats -g 1 -e # Enable stats

Base Command Manager (BCM)

cmsh # Enter CLI [cmsh] device use node001 # Manage specific node [cmsh] softwareimage list # List images [cmsh] monitoring setup # Config alerts
Head Node: Provisioning, Monitoring, User Mgmt.
Software Image: Golden image for stateless nodes.

Docker / NGC

docker login nvcr.io # Auth with API Key docker run --gpus all ... # Enable all GPUs docker run --gpus '"device=1,2"' ... # specific GPUs
Issue? Check /etc/docker/daemon.json has default-runtime: nvidia

Troubleshooting Flow

1. Hardware/Driver: nvidia-smi (fails? check dmesg, reboot)

2. K8s/Docker: can container see GPU? (docker run --gpus all ...)

3. Workload: OOM? Permissions? (CUDA_VISIBLE_DEVICES match?)

4. Performance: nvidia-smi dmon, dcgmi dmon (Throttling? Power?)
